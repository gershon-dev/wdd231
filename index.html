About This Course

WDD231 – Frontend Web Development I builds on HTML and CSS fundamentals by introducing JavaScript for dynamic web interactions, responsive design, accessibility, and performance best practices.


Key skills covered include DOM manipulation, Fetch API, local storage, JSON, and modern CSS layout techniques.
